Caesar Cipher Project

A programming project I undertook at The Developer Academy. This web app accepts a message from the user and outputs its encoded version. Built with JavaScript, the app modifies each letter of the input message by shifting it a specified number of positions in the alphabet, as determined by the user. I further enhanced the app by incorporating a decrypt function, enabling the decoding of coded messages. I utilized Tailwind for styling, achieving a clean and modern appearance.
